# Interactive Character Showcase

This repository hosts a Three.js animated GLB character.

**Live Demo:** https://mirzasnowy.github.io/snowy-3d-showcase/

## Features

- Three.js (vanilla, no build tools)
- GLTFLoader for `.glb` model
- Animation playback starts at 2 seconds
- Non-looping animation with clamped final pose
- Transparent background
- Orbit controls
- GitHub Pages deployment
